Лабораторная работа 2. Основы NumPy: массивы и векторные операции

Описание задачи

Целью лабораторной работы было изучение библиотеки NumPy и её возможностей для работы с массивами, векторными и матричными операциями, статистическим анализом данных и визуализацией.

В рамках работы требовалось реализовать набор функций для: - создания и преобразования массивов; - выполнения векторных операций; - выполнения матричных вычислений; - загрузки и анализа набора данных; - нормализации данных; - построения графиков.

Также код должен был проходить тесты, содержать аннотации типов, докстринги и соответствовать требованиям PEP-8.

Как была решена задача

Для решения лабораторной работы была использована библиотека NumPy, а также дополнительные библиотеки: - pandas — для чтения CSV-файла; - matplotlib и seaborn — для построения графиков.

Сначала были реализованы базовые функции работы с массивами: - создание одномерного массива от 0 до 9; - создание случайной матрицы 5x5; - изменение формы массива; - транспонирование матрицы.

Затем были реализованы функции векторных операций: - поэлементное сложение; - умножение вектора на число; - поэлементное умножение; - скалярное произведение.

После этого были реализованы матричные операции: - умножение матриц; - вычисление определителя; - нахождение обратной матрицы; - решение системы линейных уравнений.

Для работы с данными была реализована загрузка CSV-файла в NumPy-массив, а также функция статистического анализа, которая вычисляет: - среднее значение; - медиану; - стандартное отклонение; - минимум и максимум; - 25-й и 75-й перцентили.

Также была реализована min-max нормализация данных по формуле:

(x - min) / (max - min)

В последней части работы были реализованы функции визуализации: - гистограмма; - тепловая карта; - линейный график.

Все изображения сохраняются в папку plots.

Нюансы решения

Векторизация

Основной особенностью NumPy является возможность выполнять операции над массивами без использования циклов. Это делает код короче, понятнее и быстрее.

Например, поэлементное сложение выполняется как a + b, а не через цикл for.

Разница между видами умножения

Важно различать: - a * b — поэлементное умножение; - a @ b — матричное умножение.

Это разные операции, и выбор зависит от поставленной задачи.

Изменение формы массива

Операция reshape не изменяет сами данные, а только их представление в памяти в другой форме.

Нормализация

Min-max нормализация удобна тем, что приводит данные к диапазону от 0 до 1, что полезно для последующего анализа и машинного обучения.

Визуализация

При построении графиков важно не только отобразить их, но и корректно сохранить в файл. Также после сохранения нужно закрывать фигуру через plt.close(), чтобы не накапливать открытые окна и объекты графики.

Вывод

В ходе выполнения лабораторной работы были изучены основные возможности библиотеки NumPy для работы с массивами и матрицами, а также основы анализа и визуализации данных. Работа показала, что NumPy позволяет удобно и эффективно выполнять численные вычисления без ручной обработки элементов через циклы.
