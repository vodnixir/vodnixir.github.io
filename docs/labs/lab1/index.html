Лабораторная работа 1

Цель работы

Освоить создание и публикацию статического сайта с использованием MkDocs и GitHub Pages.

Задание

Создать сайт-портфолио, настроить структуру страниц, конфигурацию проекта и публикацию через GitHub Pages.

Ход выполнения

  1. Создан публичный репозиторий GitHub.
  2. Настроено виртуальное окружение Python.
  3. Установлен MkDocs.
  4. Создана структура сайта.
  5. Настроен файл mkdocs.yml.
  6. Выполнена сборка сайта в каталог docs.
  7. Сайт опубликован через GitHub Pages.

Код и команды

```bash mkdocs new source cd source mkdocs serve mkdocs build -d ../docs
