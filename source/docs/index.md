# Главная

Добро пожаловать на мой сайт-портфолио лабораторных работ.

На этом сайте представлены материалы по лабораторным работам курса, выполненные в рамках обучения. Для удобства навигации все работы распределены по отдельным страницам.

## Что находится на сайте

- информация об авторе;
- список лабораторных работ;
- отдельные страницы с описанием каждой работы;
- структура отчёта: цель, задание, код, выводы.

## Навигация

Используйте меню сайта для перехода между разделами.
